# Smoking Ban Policy Evaluation

Causal evaluation of comprehensive smoking bans using state-year panel data and event-study methods.

## Research Question
Do comprehensive smoking bans reduce adult smoking rates?

## Data
State-year panel constructed using:
- CDC STATE System (smoke-free laws)
- BRFSS smoking prevalence data

## Methodology
- Difference-in-differences with state and year fixed effects
- Event-study specification to test parallel trends
- Standard errors clustered at the state level

## Tools
Python, pandas, statsmodels
